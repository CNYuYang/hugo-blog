openGuass编译

2023-12-04 00:55:34
#高斯  #SQL  #开源 

下载并安装CentOS7.6：https://mirrors.aliyun.com/centos-vault/7.6.1810。这里选择的是CentOS-7-x86_64-DVD-1810.iso，避免其他版本出现缺失软件的情况。

下载编译工具：https://gitee.com/opengauss/openGauss-server#%E4%B8%8B%E8%BD%BDopengauss，需根据对应系统以及高斯版本下载。这里我安装的是CentOS7.6，选择的gcc版本为7.6，以及主线版本：

安装git，创建code目录并拉取代码：

git clone https://gitee.com/opengauss/openGauss-server

再上传上述下载的编译工具包：

解压并重命名：

# 解压
tar -xvf openGauss-third_party_binarylibs_Centos7.6_x86_64.tar.gz
# 重命名
mv openGauss-third_party_binarylibs_Centos7.6_x86_64 binarylibs

安装依赖

yum install -y libaio-devel ncurses-devel pam-devel libffi-devel python3-devel libtool libtool-ltdl openssl-devel bison flex glibc-devel patch readline-devel lsb_release

进入openGauss-server目录，执行编译脚本：

sh build.sh -m debug -3rd /home/yuyang/code/binarylibs

看到openGauss installation complete.就代表高斯编译并安装成功：

安装成功后会出现mppdb_temp_install目录，该目录下有完整的高斯结构：
